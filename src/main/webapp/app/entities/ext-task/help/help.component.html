Welcome to help page.

Terminologies used:

  • Task:
    An activity assigned to MT needs to understand the requirements, use the provided files and upload completed work.
  • File:
    Any audio or documents. Managers, editors and MT can upload and download files.
  • TaskGroup:
    Collection of tasks.

      • Important: Link user with user Info.Entities -> User Info
  • Creating tasks:
    There are two ways to create tasks.
    • Go to TaskGroup list Manager -> Task group list. Click upload files button ('Create new tasks').
      New tasks with the uploaded files will be created.
    • Go to above page, click view icon of any taskGroup. In the taskGroup detail page, click upload files button ('Add more tasks')
      New tasks are created under the current taskGroup.

Editor:

  • Once MT upload their finished documents, they will transfer control to editors.
  • Editors may go through the tasks assigned to them, download and upload files.
  • Upon successful completion of tasks, control need to be transferred to manager.
Please go through following section 'MT' to understand the process of handling tasks.

Transcript:

  • Go to "My tasks". The task page is divided into two. Active tasks and all past tasks are listed here.
  • You may download or upload files. Once tasks are completed, tasks are assigned to editors.
    • While downloading the files, make sure you "save as" the file with .zip extension
  • Clone task You may select few tasks and replicate them. Copy of the tasks are placed in the same group. Files of original task is copied to cloned task
  • Merge tasks Two or more tasks can be merged together. Given a seleted list of tasks, a resultant new task is created. All attributes of the first selected task is copied to the new task. All files of all selected tasks are copied under the new task.
